COVID-19 REST API for India

API for COVID-19 stats in India sourced from many different DataSources. This REST API is a fetches date from the data source and convert it into the required forms and return the JSON data.

API

Data URL
Case Counts

Latset case counts of COVID-19 by state

 /rootnet/api/latest/stats
Case History

Historical case counts of COVID-19 by state since 10 March 2020

 /rootnet/api/latest/stats/history
Case by Districts

Latset case counts of COVID-19 by districts of all states

 /covid19india/api/case_by_district
